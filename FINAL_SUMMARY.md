### 2. **При обновлении машины (воркер)**:
```sql

UPDATE cars SET 
  has_details = true,
  last_detail_update = NOW(),
  power = '440(598Ps)',
  image_gallery = 'url1.jpg url2.jpg url3.jpg',
  image_count = 14,
  dealer_info = 'Название: ...; Адрес: ...',
  ...
WHERE uuid = 'xxx'
```

✅ **НЕ трогает**: `source`, `car_id` (избегаем конфликтов индекса)

### 3. **Индекс исправлен**:
```sql
-- Старый (вызывал ошибки):
CREATE UNIQUE INDEX idx_cars_source_car_id ON cars(source, car_id);

-- Новый (работает корректно):
CREATE UNIQUE INDEX idx_cars_source_car_id ON cars(source, car_id) 
WHERE source != '' AND car_id != 0;
```

✅ **Частичный индекс**: Игнорирует пустые значения
