# Telegram Car App

Приложение для отображения данных об автомобилях из различных источников.

## Функциональность

- Просмотр списка автомобилей с пагинацией
- Фильтрация по источнику, бренду, городу, году и поиск по названию
- Адаптивный дизайн для мобильных и десктопных устройств

## Технологии

- React 18
- CSS3
- Fetch API для работы с бэкендом

## Запуск приложения

1. Установите зависимости:
```
npm install
```

2. Запустите приложение в режиме разработки:
```
npm start
```

3. Для сборки приложения:
```
npm run build
```

## API Endpoints

Приложение использует следующие API endpoints:

- `GET /cars` - Получение списка автомобилей с фильтрами и пагинацией
  - Параметры:
    - `page` - Номер страницы (по умолчанию: 1)
    - `limit` - Количество элементов на странице (по умолчанию: 20)
    - `source` - Фильтр по источнику (опционально)
    - `brand` - Фильтр по бренду (опционально)
    - `city` - Фильтр по городу (опционально)
    - `year` - Фильтр по году (опционально)
    - `search` - Поиск по названию (опционально)
