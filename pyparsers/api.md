# API Endpoints for pyparsers

## Synchronous API Endpoints

### `get_dongchedi_cars`
- **Description**: Fetches car data from Dongchedi.
- **Path**: Not specified in the code.
- **Method**: GET
- **Details**: Filters cars with a manufacturing year of 2017 or later.

### `get_dongchedi_cars_by_page`
- **Description**: Fetches car data from Dongchedi for a specific page.
- **Path**: Not specified in the code.
- **Method**: GET
- **Parameters**:
  - `page` (int): Page number (starting from 1).
- **Details**: Filters cars with a manufacturing year of 2017 or later.

### `get_che168_cars`

- **Path**: Not specified in the code.

- **Details**: Filters cars with a manufacturing year of 2017 or later.
